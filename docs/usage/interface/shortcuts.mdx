LobeHub offers a wide range of keyboard shortcuts to help you perform actions quickly and boost your productivity. Mastering these shortcuts can significantly reduce mouse usage and streamline your workflow.

## Accessing Settings

Click your user avatar in the top-right corner of the homepage. From the dropdown menu, select "App Settings," then click on "Shortcuts" to view all available keyboard shortcuts.

!\[Image]

## Basic Shortcuts

- Open the global command palette for quick access: ⌘ + K (Mac) or Ctrl + K (Windows/Linux)
- Focus the main search bar on the current page: ⌘ + J (Mac) or Ctrl + J (Windows/Linux)
- Quickly switch between assistants: Hold Ctrl and press a number key (0–9) to switch to a pinned assistant in the sidebar: ^ + 1–9 (Mac) or Ctrl + 1–9 (Windows/Linux)
- Switch to the default conversation: ^ + 0 (Mac) or Ctrl + 0 (Windows/Linux)
- Show/Hide the left panel: ⌘ + \[ (Mac) or Ctrl + \[ (Windows/Linux)
- Show/Hide the right panel: ⌘ + ] (Mac) or Ctrl + ] (Windows/Linux)
- Open shortcut help: ^ + ⇧ + ? (Mac) or Ctrl + Shift + ? (Windows/Linux) to view all shortcut instructions.
- Toggle Focus Mode: ⌘ + \ (Mac) or Ctrl + \ (Windows/Linux). In Focus Mode, only the current conversation is displayed, with other UI elements hidden.
- Save document: ⌘ + S (Mac) or Ctrl + S (Windows/Linux)

## Conversation Shortcuts

- Open conversation settings: ⌘ + , (Mac) or Ctrl + , (Windows/Linux) to view and modify current conversation settings.
- Regenerate the last message: ⌘ + R (Mac) or Ctrl + R (Windows/Linux)
- Delete the last message: ⌘ + D (Mac) or Ctrl + D (Windows/Linux)
- Delete and regenerate the last message: ⌘ + ⇧ + R (Mac) or Ctrl + Shift + R (Windows/Linux)
- Save the current topic and start a new one: ⌘ + N (Mac) or Ctrl + N (Windows/Linux)
- Add a user message: ⌘ + ↵ (Mac) or Ctrl + Enter (Windows/Linux) to add the current input as a user message without triggering a response.
- Edit a message: ⌘ + ⌥ (Mac) or Ctrl + Alt (Windows/Linux). Hold Alt and double-click a message to enter edit mode.
- Clear all messages in the conversation: ⌘ + ⇧ + ⌫ (Mac) or Ctrl + Shift + Backspace (Windows/Linux)

## Customizing Shortcuts

Most shortcuts can be customized to suit your workflow. Tailor the most frequently used actions to your preferences for a smoother experience. In the shortcut settings panel, click on any editable shortcut button and press your desired key combination to record a custom shortcut.
