# 提交你的助理

如果你创建了优质的助理，可以提交到市场与全球用户分享。

如果你是首次提交助理，需要先创建社区个人档案，以便在社区上提交和管理上架信息。

点击左侧边栏的「社区」→「助理」→右上角「成为创作者」。

\[Image]

当你已成为创作者，选择需要发布的助理进入「助理档案」，点击「分享助理到社区」即可。

\[Image]

通过审核后，你的助理会出现在市场中，供全球用户使用。LobeHub 欢迎所有用户加入这个不断成长的生态系统，共同参与到助理的迭代与优化中来。共同创造出更多有趣、实用且具有创新性的助理，进一步丰富助理的多样性和实用性。
