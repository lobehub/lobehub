# Using Zidong Taichu in LobeChat

<Image alt={'Using Taichu in LobeChat'} cover src={'https://github.com/user-attachments/assets/9cb27b68-f2ac-4ff9-8f97-d96314b1af03'} />

This guide will walk you through how to use Zidong Taichu in LobeChat:

<Steps>
  ### Step 1: Obtain a Zidong Taichu API Key

  - Create a [Zidong Taichu](https://ai-maas.wair.ac.cn/) account
  - Generate and retrieve your [API Key](https://ai-maas.wair.ac.cn/#/settlement/api/key)

  <Image alt={'Create API Key'} inStep src={'https://github.com/user-attachments/assets/8d90ae64-cf8e-4d90-8a31-c18ab484740b'} />

  ### Step 2: Configure Zidong Taichu in LobeChat

  - Go to the LobeChat `Settings` page
  - Under `AI Providers`, locate the `Zidong Taichu` configuration section

  <Image alt={'Enter API Key'} inStep src={'https://github.com/user-attachments/assets/55028fe5-44db-49e2-93c5-5dabbd664f10'} />

  - Paste the API Key you obtained
  - Choose a Zidong Taichu model for your AI assistant to start chatting

  <Image alt={'Select Taichu model and start chatting'} inStep src={'https://github.com/user-attachments/assets/c44b6894-70cb-4876-b792-2e76e75ac542'} />

  <Callout type={'warning'}>
    You may need to pay for API usage depending on the provider’s pricing policy. Please refer to Zidong Taichu’s official pricing for details.
  </Callout>
</Steps>

Now you're all set to start using Zidong Taichu models for conversations in LobeChat.
