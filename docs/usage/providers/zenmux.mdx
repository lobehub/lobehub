# Using ZenMux in LobeChat

Welcome to **[ZenMux](https://docs.zenmux.ai/en/about/intro.html)**! This guide will help you get started quickly.

::: tip ðŸ’¡ Get Started in 4 Steps
You can start using ZenMux in just four simple steps:
:::

<Steps>
  ### Step 1: Login to ZenMux

  Visit the **[ZenMux Login Page](https://zenmux.ai/login)** and choose one of the following methods to log in:

  - Email Login
  - GitHub Account Login
  - Google Account Login

  ### Step 2: Get API Key

  After logging in, go to your **[User Console > API Keys](https://zenmux.ai/settings/keys)** page and create a new API Key.

  ### Step 3: Configure LobeChat

  - Access the `Settings` interface in LobeChat
  - Find `ZenMux` under `AI Service Provider`
  - Turn on ZenMux and enter the obtained API Key

  ### Step 4: Fetch Latest Models

  - As shown in the figure, click the `Get Model List` button to fetch the latest models

  ![zenmux-lobechat](https://cdn.marmot-cloud.com/storage/zenmux/2025/12/29/OWF957X/lobe-chat.png)
</Steps>
