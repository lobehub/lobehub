# Using Volcengine in LobeChat

<Image alt={'Using Volcengine in LobeChat'} cover src={'https://github.com/user-attachments/assets/b9da065e-f964-44f2-8260-59e182be2729'} />

[Volcengine](https://www.volcengine.com/) is a cloud service platform under ByteDance. Through its "Volcano Ark" platform, it provides large language model (LLM) services, supporting several mainstream models such as Baichuan Intelligence and Mobvoi.

This guide will walk you through how to use Volcengine in LobeChat:

<Steps>
  ### Step 1: Obtain a Volcengine API Key

  - First, visit the [Volcengine official website](https://www.volcengine.com/) and complete the registration and login process.
  - Navigate to the [Volcano Ark Console](https://console.volcengine.com/ark/)

  <Image alt={'Accessing the Volcano Ark API management page'} inStep src={'https://github.com/user-attachments/assets/d6ace96f-0398-4847-83e1-75c3004a0e8b'} />

  - Go to the `API Key Management` section and click `Create API Key`
  - Copy and securely save your newly created API Key

  ### Step 2: Configure Volcengine in LobeChat

  - Open the `App Settings` in LobeChat and go to the `AI Service Providers` section
  - Find the `Volcengine` option in the list of providers

  <Image alt={'Enter your Volcengine API Key'} inStep src={'https://github.com/user-attachments/assets/237864d6-cc5d-4fe4-8a2b-c278016855c5'} />

  - Enable the Volcengine provider and paste in your API Key
  - Choose a Volcengine model for your assistant to start chatting

  <Image alt={'Select a Volcengine model'} inStep src={'https://github.com/user-attachments/assets/702c191f-8250-4462-aed7-accb18b18dea'} />

  <Callout type={'warning'}>
    You may incur charges when using the API services. Please refer to Volcengineâ€™s pricing policy for details.
  </Callout>
</Steps>

You're now ready to start chatting with models powered by Volcengine in LobeChat.
