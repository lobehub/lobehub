The Command Menu is LobeHub’s shortcut action center. Instantly accessible via keyboard shortcuts, it allows you to search and execute various actions without navigating through multiple interface layers. This makes navigation and operations faster and more efficient.

## Opening the Command Menu

Use the following keyboard shortcuts to open the Command Menu:\
macOS: `Cmd + K`\
Windows/Linux: `Ctrl + K`\
The Command Menu will appear as a floating overlay in the center of your screen.

!\[Image]

## Search and Navigate

Type keywords into the Command Menu to instantly search for matching content. You can search for assistant names to quickly switch conversations, enter topic keywords to jump to past chats, access sections like Settings, Documents, and Library, or search for plugins and files. The Command Menu also supports command-based actions, allowing you to quickly create assistants, start AI conversations, and more.

!\[Image]

## Keyboard Navigation

Navigate the Command Menu using your keyboard:\
Use the ↑ and ↓ arrow keys to move through search results,\
Press Enter to execute the selected action,\
Press Esc to close the Command Menu.

## Tips & Tricks

The Command Menu supports fuzzy search, so you don’t need to type full names—partial keywords will do. For example, typing "code" can bring up "Python Coding Assistant."\
If you frequently switch between assistants, using the Command Menu is much faster than browsing the sidebar.\
Memorizing a few common keywords can significantly boost your productivity.

## Use Cases

### Chat with an Assistant

Press `Cmd + K` to open the Command Menu, type your topic, press Tab to select an assistant, and start chatting instantly.

### Quick Access to Other Sections

Press `Cmd + K`, use the ↑ and ↓ keys to navigate the menu, select a section like "Documents" or "Resources," and press Enter to open it.

### Create an Assistant or Group Chat

Press `Cmd + K`, use the ↑ and ↓ keys to find "New Assistant" or "New Assistant Team," and press Enter to create.

### Search for Information

Press `Cmd + K`, enter a keyword to search through past messages, assistants, MCP plugins, and more.\
Search results are categorized for easy browsing.
